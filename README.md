# 🔥 BarberPro 2.0

Sistema completo de gestão para barbearias com WhatsApp IA.

## 🚀 Stack

- Frontend: React + TypeScript + Vite + Tailwind
- Backend: Node + Express + Prisma + PostgreSQL
- WhatsApp: Baileys (gratuito!)
